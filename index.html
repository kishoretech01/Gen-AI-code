Your GenAI Journey: From Python to Projects

Each week brings new skills and a unique project idea tailored just for you. No two students get the same challenge. This is deep, industry-ready training ‚Äî not a shortcut.

Week 1-2: Python Foundation

Master Python syntax, control flow, data structures, functions, and file handling. Receive your first project idea based on your strengths.

Week 3: Intermediate Python + Libraries

Work with OOP, NumPy, Pandas, and visualize data using Matplotlib/Seaborn. Clean and analyze real datasets.

Week 4: Machine Learning & AI Basics

Understand AI vs ML vs DL. Use Scikit-learn to build models like regression, classification, and clustering with real-world data.

Week 5: Generative AI & Prompt Engineering

Explore LLMs (GPT, Gemini), prompt writing, embeddings, and start working with OpenAI API for text/code generation. One-on-one project brainstorming session.

Week 6: RAG, Vector DBs, and Chatbots

Build a mini RAG project using ChromaDB/FAISS. Learn how to use vector embeddings in real applications. Apply everything in a guided project.

Week 7: Final Project Execution

Develop a unique end-to-end project ‚Äî like a GenAI chatbot, Resume Ranker, or PDF QA bot. Projects are assigned based on your performance. No repetition across students.
